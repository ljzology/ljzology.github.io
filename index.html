博客搭建踩坑记录

参考文档

GitHub windows客户端简单使用教程
手把手教你使用Hexo + Github Pages搭建个人独立博客
在朋友的帮助下少踩了很多坑，博客搭建的还算顺利

步骤

搭建环境：win7

1.Git bash

按照文档1安装完github客户端后，主要在Git bash进行命令行操作

2.Hexo

Hexo 是一个简单地、轻量地、基于Node的一个静态博客框架,可以方便的生成静态网页托管在github上。

踩到的坑

数学公式不能被正常渲染

按照文档在Hexo中渲染MathJax数学公式操作后，还是不行。
在Google查找原因后，安装了两个工具：pandoc、latex后，问题成功解决。
更多的踩坑记录请参考朋友博客
